# ai-skills

Коллекция специализированных навыков для Claude AI, организованная для расширения возможностей ассистента в различных технических и профессиональных областях.

## О проекте

**ai-skills** — это хранилище структурированных расширений (skills) для Claude, которые включают:

- **Специализированные знания** — глубокие инструкции и примеры для сложных задач
- **Триггеры активации** — ключевые слова на английском и русском языках для автоматического вызова навыков
- **Готовые к использованию шаблоны** — стандартизированные подходы и workflows для решения типичных проблем
- **Метаданные навыков** — чёткое определение области применения каждого навыка через YAML-фронтматтер

## Структура проекта

```
.claude/
└── skills/
    ├── 3d-modeling/          # Создание 3D-моделей, CAD, 3D-печать
    ├── create-prompt/        # Создание промтов и инструкций для агентов
    ├── decompose-task/       # Декомпозиция задач и планирование шагов
    └── task-writer/          # Формирование задач и тикетов для трекера
```

## Навыки

### 3D Modeling (`3d-modeling/`)

Навык для создания профессиональных 3D-моделей программным способом:

- **Форматы**: STL (для 3D-печати), STEP (для CAD), 3MF, IGES, OBJ
- **Возможности**: создание механических деталей, резьбы, шестерёнки, сборки, операции лофт/свип/револв
- **Стек**: Python + CadQuery + OpenCASCADE
- **Триггеры**: STL, STEP, IGES, CAD, 3D print, резьба, шестерёнка, крышка, деталь, сборка и т.д.

### Create Prompt (`create-prompt/`)

Навык для формирования чётких и эффективных промтов для агентов:

- **Цели**: генерация структурированных инструкций, шаблонов и систем промтов
- **Триггеры**: prompt, промт, template, instruction, агент, задачу

### Decompose Task (`decompose-task/`)

Навык для разбивки сложных задач на последовательные шаги:

- **Цели**: создание планов, workflow, поэтапной декомпозиции
- **Триггеры**: план, разбить задачу, декомпозиция, этапы, workflow

### Task Writer (`task-writer/`)

Навык для описания задач и тикетов в трекере:

- **Цели**: написание технических требований, описаний, acceptance criteria
- **Триггеры**: задача, тикет, ТЗ, описание задачи, оформить задачу

## Использование

Каждый навык находится в отдельной папке под `.claude/skills/` с файлом `SKILL.md`, содержащим:

1. **YAML-метаданные** (в начале файла) — название, описание, триггеры активации
2. **Detailed Instructions** — пошаговые инструкции, примеры кода, стандартные workflows
3. **Reference Materials** — ссылки на документацию, таблицы данных, стандарты

## Как расширить

Для добавления нового навыка:

1. Создайте папку в `.claude/skills/[название-навыка]/`
2. Добавьте файл `SKILL.md` с YAML-фронтматтером и описанием
3. Определите триггеры активации (ключевые слова на EN и RU)
4. Напишите подробные инструкции и примеры использования
